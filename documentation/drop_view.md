# Drop View #

Drops an existing view.

## Sample ##

{% highlight xml %}
<dropView viewName="personView"/>
{% endhighlight %}

## Available Attributes ##

^ viewName  | Name of the view to drop **required**  |
^ schemaName  | Name of the view schema  |
